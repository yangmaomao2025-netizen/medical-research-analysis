## 一、腾讯觅影（Tencent Miying）

### 1.1 基本信息
- **官网：** tencentmiying.com
- **定位：** 医疗影像AI + 临床辅助决策平台
- **核心团队：** 50+博士AI科学家，400+AI应用工程师

### 1.2 产品矩阵

#### AI产品
| 产品名称 | 功能描述 |
|---------|---------|
| **AI临床助手** | 基于医学知识图谱，用药风险审核（8w+药物），Top5推荐准确率>96% |
| **智能预问诊** | 全科预问诊支持，口语化症状转译，电子病历一键生成 |
| **智能导诊** | 基于NLP技术的智能导诊系统 |
| **AI精准预约** | 患者病情智能判病，医患精准匹配 |
| **AI就医助手** | 智能对话机器人 |
| **肺炎智能评估** | 肺部CT影像AI分析 |

#### 云产品
| 产品名称 | 功能 |
|---------|------|
| **病理云** | 病理切片存储、分析、检索 |
| **影像云** | 医学影像服务，分级诊疗支持 |

#### 平台产品
| 产品名称 | 功能 |
|---------|------|
| **开放实验平台** | 基于腾讯云的医学AI开放创新服务平台 |

### 1.3 技术能力
- **GPU集群：** 价值数亿，单检查500+张图片，6秒完成AI运算
- **数据安全：** 物理隔离存储，加密传输，与医院签订保密协议
- **医学知识：** 联合人民卫生出版社等权威机构

### 1.4 应用场景
- 区域基层医疗机构
- 云HIS诊所
- 各级单体医院
- 区域卫生监管

### 1.5 优势
✅ 影像AI能力强（腾讯在图像识别领域积累深厚）
✅ 产品矩阵完整（从导诊到临床决策全覆盖）
✅ 数据安全保障完善
✅ 与顶级医学院校合作

### 1.6 劣势
❌ 主要聚焦影像和西医，**中医领域较少涉及**
❌ 基层医疗覆盖为主，三甲医院深度不足
❌ 病历内涵质控能力未明确展示

## 二、百度灵医智惠（Baidu Lingyi）

### 2.1 基本信息
- **愿景：** "循证AI赋能基层医疗"
- **技术基础：** 百度大脑、飞桨深度学习平台
- **核心特点：** **循证医学驱动**

### 2.2 产品系列

| 产品名称 | 功能描述 |
|---------|---------|
| **临床辅助决策系统（CDSS）** | 基于循证AI的诊疗建议 |
| **眼底影像分析系统** | 眼底病变AI筛查 |
| **医疗大数据整体解决方案** | 数据治理、深度结构化 |
| **智能诊前助手** | 诊前信息采集、分诊 |
| **慢病管理平台** | 慢病全程管理 |
| **灵医Bot** | 医疗健康大模型（2026年发布） |

### 2.3 核心能力

#### 循证AI特点
- **符合医学事实：** 内容严谨，有证可循
- **安全性：** 保障用户信息安全、诊疗安全
- **逻辑性：** 逻辑通顺，逼近真人推理
- **理解力：** 准确理解用户意图
- **完整性：** 内容全面呈现

#### 数据治理能力
- 专病深度结构化（胃癌、乳腺癌、肺癌等）
- 准确率和召回率均达**96%以上**
- 每例患者数据整理时间节约**30-40分钟**

### 2.4 客户案例
- 南昌大学第一附属医院
- 北大医疗产业集团
- 近千家医疗机构

### 2.5 最新动态（2026年1月）
- 与**固生堂**达成战略合作
- 联合推动医学行业大模型产业化
- **国家医学考试中心**合作项目

### 2.6 优势
✅ **循证医学理念**（与我们方向一致）
✅ 通过国家级医学考试验证
✅ 数据治理能力突出
✅ 飞桨平台全流程自主可控

### 2.7 劣势
❌ 主要聚焦**基层医疗**
❌ **中医特色不明显**
❌ 病历内涵质控非核心卖点
❌ 与三甲医院HIS系统深度集成案例较少

## 六、市场机会分析

| 空白领域 | 竞品现状 | 我们的机会 |

### 6.2 我们的差异化定位

```
腾讯觅影：    影像AI + 通用临床辅助

讯飞医疗：    语音交互 + 便捷录入

```
