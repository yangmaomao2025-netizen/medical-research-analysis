# Tailwind CSS Documentation Skill

Complete Tailwind CSS documentation packaged as an OpenClaw AgentSkill.

## Contents

- **All Utility Classes** (185 utility documentation files)
- **Core Concepts** (installation, responsive design, dark mode)
- **Customization** (configuration, theming, plugins)
- **Layout & Spacing** (flexbox, grid, positioning)
- **Typography** (fonts, text utilities)
- **Backgrounds & Borders** (colors, gradients, shadows)
- **Effects & Filters** (transitions, animations, transforms)
- **Responsive Design** (breakpoints, modifiers)

## Installation

Via ClawHub:
```bash
clawhub install lb-tailwind-skill
```

Or manually: Download and extract into your OpenClaw workspace `skills/` folder.

This skill triggers automatically when you ask questions about Tailwind CSS utilities, responsive design, dark mode, configuration, or styling patterns.
